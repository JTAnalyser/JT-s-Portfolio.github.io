Regression using python

For this project, four different prediction models using python for the prediction of energy consumption in a building were developed. I developed a linear regression model, random forest regression model, support vector machine regression model, and a gradient boosting regression model. There were two different approaches taken due to the complicated nature of the datasets which was subject to the huge variations in the timeline of data available. The abscence of reliable data during Covid meant that the datasets had to be divided into "Peri-Covid and Post-Covid" datasets. This helped to create a different dimension to predictive model building and a more balanced result.
