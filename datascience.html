Regression using python

For this project I built four different prediction models using python for the prediction of energy consumption in a building. I built a linear regression model, random forest model, support vector machine regression model, and a gradient boosting regression model. This is the first approach that was taken as the dataset was very complicated due to the huge variations in the timeline of data available. The abscence of reliable data during Covid meant that the datasets had to be divided into "Covid and Post-Covid" datasets.
